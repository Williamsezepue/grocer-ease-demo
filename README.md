# Grocer Ease — Supermarket E‑Commerce Demo

A clean supermarket e‑commerce demo built with **Next.js** and **Tailwind CSS**.  
Grocer Ease features dynamic product catalogs, cart and checkout flows, and responsive layouts tailored for grocery shopping.  
Designed for performance, accessibility, and clarity to enhance online grocery experiences.

---

## ✨ Features

- Dynamic product catalog with categories for fresh produce, packaged goods, and household items
- Shopping cart and checkout flow optimized for speed and clarity
- Responsive layouts for mobile and desktop grocery shopping
- Modern animations and transitions for engaging user experience
- Accessibility‑first design with SEO‑friendly structure

---

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Framer Motion](https://www.framer.com/motion/)
- [Supabase](https://supabase.com/)
